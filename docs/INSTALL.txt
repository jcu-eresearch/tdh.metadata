tdh.metadata Installation
-------------------------

To install tdh.metadata using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``tdh.metadata`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        tdh.metadata
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
